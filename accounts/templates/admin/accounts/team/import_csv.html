Import teams from CSV

Upload a CSV file with one team per row. Required columns: name and password. Passwords are read in plaintext and hashed on save — delete the source file after import.

Comma-delimited and semicolon-delimited files are both accepted. Existing teams in the chosen hunt are kept; the new rows are added on top. If any row fails validation, nothing is imported.
